Tool 01

BecaGraph

CIDOC CRM Entity Resolution Neo4j
From Spreadsheet
to Knowledge Graph
Data Integration Engine · The Foundation

BecaGraph is the integration engine that powers the entire platform. It transforms raw Excel or CSV exports from museum collection management systems into a CIDOC CRM-compliant Neo4j knowledge graph — with LLM-powered entity resolution that identifies the same actor, place, or object across multiple institutional datasets.

📂
Upload objects, actors, events, and institutions from any museum data export (XLSX · CSV)
🧠
Semantic matching resolves duplicate entities across institutions at configurable thresholds
🗺️
CIDOC CRM classes mapped — E22 Objects, E39 Actors, E8 Acquisitions, E53 Places…

Tool 02

BecaChat

Conversational AI Live Cypher
Ask the Graph,
Get Answers
Natural Language Knowledge Graph Query

BecaChat lets researchers and museum professionals query the BECACO knowledge graph in plain language — no Cypher query language expertise needed. Type a question, and the LLM translates it into a graph query, executes it against Neo4j, and returns a structured, readable answer.

🗣️
Accepts questions in plain English — "Who acquired this object from whom and when?"
⚡
Automatically generates and executes Cypher queries against the live Neo4j graph
🔍
Returns structured, explainable answers with provenance chain details
Who supplied objects from Mexico to European museums between 1950 and 1980?
BecaChat · Graph Query
MATCH (a:Actor)-[:SUPPLIED]->(o:Object)
WHERE o.culture = "Mexico"
AND o.acquisition_year >= 1950
RETURN a.name, count(o) ORDER BY count(o) DESC

Tool 03

BecaNarratives

Story Generation Text-to-Speech Multi-Modal
Objects with
Stories to Tell
Object Story Weaver · LLM Narrative Engine

Beca Narratives bridges the gap between raw provenance data and human-readable stories. Enter any object identifier from the knowledge graph, and the tool retrieves its structured metadata, feeds it to an LLM, and generates a compelling, historically informed narrative. The text is then converted to audio for accessible, multi-modal playback — validated by both domain experts and museum visitors.

🔍
Autocomplete search across all object identifiers in the graph
📜
Structured metadata panel + LLM-generated narrative displayed side by side
🔊
Text-to-speech playback with play/stop controls — accessible to non-specialist audiences
